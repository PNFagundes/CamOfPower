# Real-Time Human Segmentation and Background Replacement in Video Streams

**Objective:** Develop an efficient computer vision method to identify individuals in live video streams and seamlessly replace them with the background;

**Methodology:** Utilized HSV color space manipulation for precise color segmentation, applying morphological operations to enhance object boundaries. Implemented logic for background capture and dynamic mask updating;

**Result:** Successfully created a real-time "invisibility" application using classical image processing techniques;
